College Application

Technology Stack

  • Frontend: HTML, CSS, JSP
  • Backend: Java (Spring for business logic)
  • Database: MySQL
  • Server: Tomcat 10.1
  • Authentication: Session-based login/signup

Project Overview

A web-based system where teachers can take attendance and update student marks, while students can view their attendance percentage and marks.

Key Features

User Authentication

  • Separate login/signup for students and teachers
  • Session-based authentication using JSP

Teacher Dashboard

  • View and manage students by branch
  • Take attendance and update marks
  • Filter and search students

Student Dashboard

  • View attendance percentage
  • View subject-wise marks
  • Check overall progress

Admin Panel (Optional Future Scope)

  • Manage teachers and students
  • Generate reports
